File format

First row is the header. Column names are case-insensitive; only name is required. Rows already in your library (by name) are skipped, so re-importing an updated file is safe.

ColumnMeaning
nameProduct name, e.g. "Simpson LUS26 Joist Hanger" (required)
categoryLibrary group key or label, e.g. "connectors" or "Connectors & Hardware"
species / grade / dimensionBrand, series, and what it fits; shown on the material card
input_typeeach (default), box, or ft
quantity_per_boxRequired for box items
lengthsFor ft items: stock lengths in feet, e.g. "8;10;12;16"
default_lengthWhich stock length quantities default to (first length if omitted)

Example row: Simpson LUS28 Joist Hanger, connectors, Simpson Strong-Tie, LUS, 2x8, each
